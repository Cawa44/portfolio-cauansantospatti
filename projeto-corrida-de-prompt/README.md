## 1. Objetivos da Atividade

- **Análise Crítica:** Nesta atividade, não somos apenas usuários de Inteligência Artificial, mas engenheiros de instrução. O objetivo é dominar a técnica de comunicação com modelos de linguagem (LLMs) para obter resultados de alta precisão no menor tempo possível.
- **Aprender a Aprender:** Desenvolver a metacognição ao identificar o que funciona ou não em cada iteração, ajustando o prompt com base no feedback da IA até atingir o resultado desejado.

---

## 2. A Tarefa

Utilizando o modelo **Qwen Chat** (https://qwen.ai/) via celular, em trio, o objetivo foi gerar dois tipos de resultado:

- **Imagem:** Um astronauta estilo barroco tocando violoncelo em Marte.
- **E-mail:** Um e-mail formal de desculpas de um pirata para um rei.

O grande desafio: atingir a perfeição com no máximo **5 iterações (tentativas)**.

---

## 3. Protocolo de Execução

### Dinâmica do Jogo

1. **A Largada:** O professor forneceu o "Alvo" — a imagem e o e-mail descritos acima.
2. **O Ciclo de Refinamento:** Para cada resposta gerada pela IA, o trio realizou duas ações antes de enviar o próximo prompt:
   - **Avaliar:** Analisar criticamente a resposta. O que faltou? O tom estava correto? A IA ignorou algum detalhe?
   - **Integrar:** Conectar o erro ao novo aprendizado, ajustando o próximo prompt com base no feedback obtido.

### Regras de Ouro

- **Limite de 5 prompts:** Cada interação conta. Caso o objetivo não fosse atingido na 5ª tentativa, o trio deveria parar e analisar o que deu errado.
- **Registro de Evolução:** Cada prompt enviado e resposta recebida foram anotados para a discussão final.

---

## 4. Registro de Performance

### Tarefa 1 — Imagem

**Dificuldades encontradas:**

Ao utilizar o Qwen Chat para gerar a imagem do astronauta estilo barroco tocando violoncelo em Marte, nos deparamos com uma limitação técnica: o modelo aparentemente não conseguiu reproduzir a Sinfonia de Beethoven como elemento visual integrado à cena. Além disso, em algumas tentativas não foi possível posicionar a câmera de forma específica, sendo necessário adaptar o prompt para solicitar um ângulo mais "natural", a fim de obter um resultado esteticamente coerente.

---

### Tarefa 2 — E-mail

**Dificuldades encontradas:**

O principal desafio foi adaptar a linguagem da época de forma que o e-mail fizesse sentido narrativo — ou seja, que tivesse coerência interna sem perder o tom histórico. Algumas respostas iniciais resultaram em textos sem coesão ("sem pé nem cabeça"), o que exigiu refinamentos no prompt para construir uma "história" que contextualizasse o pedido de desculpas de forma crível e estruturada.
